How this is scored, and how the band is set

Each of the eight scored domains carries a preference weight for the answer chosen. The weights are summed and converted with the ASCOT-SCT4 formula SCRQoL = 0.203 × weighted score − 0.466, anchored to the ideal state at 1.00. The dignity filter question is asked but not scored. The action band is separate from this score: it reads the worst domain state and the spread, not the composite, so a single high-level need is not masked by a good average. Band logic is Assistiv operational triage over ASCOT-native states, configurable and for validation with clinical advisers, not a clinical cut-off.
