# IMPORTANT, HOW THIS FILE WORKS
# A crawler obeys ONLY the most specific User-agent group that matches it.
# It does NOT also read the "User-agent: *" group. The previous version of
# this file gave each AI crawler its own group containing "Allow: /" and
# nothing else, which meant every Disallow in the "*" group was ignored by
# those crawlers. The exclusion list is therefore repeated in every group.
# If you add a new path to one group, add it to all of them.
#
# NOTE: robots.txt is advisory. It asks well-behaved crawlers not to fetch.
# It does not make anything private. Anything served by GitHub Pages remains
# publicly downloadable by anyone with the URL.
